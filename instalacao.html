Instalação do Claude Code
ANTES DE TUDO

Como instalar o Claude Code no VS Code.

Siga cada passo na ordem. Do zero ao Claude Code funcionando leva menos de 10 minutos.

REQUISITO MÍNIMO

VS Code versão 1.98.0 ou mais recente. Se já tiver instalado, confira em Help → About.

1

Instale o VS Code

Se ainda não tiver, baixe e instale o VS Code. É gratuito e funciona no Mac, Windows e Linux.

// Baixe o instalador para o seu sistema

code.visualstudio.com

Já tem instalado? Pula para o passo 2.

2

Crie uma conta na Anthropic

Acesse claude.ai, clique em "Sign up" e crie sua conta. Pode usar o Google para facilitar.

// Acesse pelo navegador

claude.ai → Sign up

Já tem conta? Pula para o passo 3.

3

Assine o plano Pro

O Claude Code precisa de uma assinatura ativa. Dentro do claude.ai, vá em configurações e assine o plano Pro.

// Caminho dentro do site

claude.ai → ícone do perfil → Upgrade plan → Pro

Já tem o plano Pro ou Max? Pula para o passo 4.

4

Abra as extensões no VS Code

Com o VS Code aberto, acesse o painel de extensões pelo ícone de peças na barra lateral esquerda ou pelo atalho de teclado.

// Atalho de teclado

Mac: Cmd + Shift + X

Windows / Linux: Ctrl + Shift + X

5

Pesquise e instale o Claude Code

Na barra de busca das extensões, digite Claude Code. Vai aparecer mais de um resultado — instale o publicado pela Anthropic.

// Verifique o publicador antes de instalar

Nome: Claude Code

Publicador: Anthropic

Clique em Install e aguarde a instalação terminar.

6

Reinicie o VS Code se necessário

Depois da instalação, o ícone do Claude Code deve aparecer na barra lateral. Se não aparecer, reinicie o VS Code.

// Recarregar sem fechar o VS Code

Cmd+Shift+P (Mac) ou Ctrl+Shift+P (Windows) → "Developer: Reload Window"

7

Clique no ícone do Claude Code

Na barra lateral esquerda do VS Code vai aparecer um ícone de faísca (Spark). Clique nele para abrir o painel do Claude Code.

// Atalho alternativo

O ícone também aparece no canto superior direito do editor quando um arquivo está aberto.

8

Faça login e pronto

Na primeira vez que abrir o painel, ele vai pedir para fazer login. Entre com a mesma conta Anthropic que você criou no passo 2. Autorize as permissões que aparecerem.

// Claude Code instalado e funcionando

Agora volte para o guia de instalação do PostPilot e continue do passo 4.

SE TRAVAR EM ALGUM PASSO

O ícone do Claude Code não aparece na barra lateral.

Abra o gerenciador de extensões, confira se o Claude Code está listado como instalado e ativo. Se estiver desabilitado, clique em "Enable". Depois recarregue o VS Code.

Apareceu mais de um "Claude Code" na busca.

Instale apenas o publicado pela Anthropic. Os outros são extensões de terceiros não oficiais.

O login não funciona ou dá erro de permissão.

Confira se a assinatura Pro está ativa em claude.ai. Sem assinatura ativa o login no Claude Code não funciona.

Meu VS Code é antigo, o que faço?

O Claude Code requer a versão 1.98.0 ou superior. Vá em Help → Check for Updates para atualizar.

PRÓXIMO PASSO

Claude Code instalado?

Agora volte para o guia principal e continue do passo 4 — clonar o repositório do PostPilot e fazer o setup da sua marca.

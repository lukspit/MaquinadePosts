Guia de Instalação
PASSO A PASSO

Do zero ao primeiro carrossel.

Siga cada passo na ordem. Do início ao fim leva menos de 15 minutos.

1

Instale o VS Code

O VS Code é o editor de código onde o sistema vai rodar. Se já tiver instalado, pula para o passo 2.

// Acesse e baixe o instalador para o seu sistema

code.visualstudio.com

2

Instale a extensão Claude Code

Com o VS Code aberto, vá até a aba de extensões (ícone de quadradinhos na barra lateral esquerda) e pesquise por Claude Code. Clique em instalar.

// Ou instale direto pelo link

marketplace.visualstudio.com → pesquise "Claude Code"

3

Assine o plano Pro da Anthropic

O Claude Code precisa de uma assinatura ativa da Anthropic. Se já tiver o plano Pro, pula para o passo 4.

// Crie sua conta ou faça login

claude.ai → planos → Pro

4

Clone o repositório do PostPilot

Abra o terminal do seu computador e rode o comando abaixo. Isso vai baixar todos os arquivos do sistema para a sua máquina.

// Cole no terminal e pressione Enter

git clone https://github.com/lukspit/CarrosseisAutom-ticos.git

Não tem o Git instalado? Baixe em git-scm.com e repita o comando.

5

Abra a pasta no VS Code

No VS Code, vá em File → Open Folder e selecione a pasta CarrosseisAutom-ticos que foi criada no passo anterior.

// Importante: abra a pasta, não um arquivo dentro dela

VS Code → File → Open Folder → CarrosseisAutom-ticos

6

Faça login no Claude Code

O painel do Claude Code vai aparecer na barra lateral. Clique em Sign In e entre com a conta Anthropic do passo 3. Se ele pedir para confirmar permissões, confirme.

// Dica

Se o painel não aparecer, use o atalho Ctrl+Shift+P (ou Cmd+Shift+P no Mac) e pesquise "Claude Code"

7

Digite /começar e responda as perguntas

No chat do Claude Code, escreva /começar e pressione Enter. O sistema vai se apresentar e começar o setup da sua marca fazendo perguntas em português. Responda tudo que ele pedir.

// O que ele vai perguntar

Nome da marca, nicho, público-alvo, cor principal, fonte, tom de voz.

Isso leva uns 5 minutos. Seja específico nas respostas — quanto melhor o input, melhor o carrossel.

8

Aguarde o setup terminar

O Claude vai instalar as dependências e configurar tudo automaticamente. Você vai ver o progresso no chat. Não precisa fazer nada, só esperar. Leva de 2 a 5 minutos dependendo da sua conexão.

// Se aparecer alguma pergunta de confirmação

Responda "sim" ou "y" — são só confirmações de instalação normais.

9

Peça o seu primeiro carrossel

Setup concluído. Agora é só falar o tema. O sistema escreve o copy, monta o design com a sua marca e entrega os slides prontos.

// Exemplos de como pedir

cria um carrossel sobre produtividade para empreendedores

cria um carrossel sobre os 5 erros mais comuns de iniciantes

/carrossel como aumentar seguidores no Instagram

DÚVIDAS COMUNS

O setup precisa ser feito toda vez?

Não. O /começar é só na primeira vez, ou quando você quiser atualizar as informações da sua marca. Depois disso, é só pedir o carrossel.

Onde ficam os slides prontos?

Na pasta output/ dentro do projeto. Cada carrossel fica numa subpasta com o tema. Os arquivos são PNG 1080x1350, prontos para subir no Instagram.

Quero mudar algo da minha marca depois.

Digite /marca no chat do Claude Code. Ele vai te fazer as perguntas de novo e atualizar o perfil da sua marca. Todos os carrosséis gerados depois disso vão refletir as mudanças.

Funciona no Windows?

Sim. O sistema detecta automaticamente o seu sistema operacional durante o setup e instala tudo da forma certa. Mac, Windows e Linux.

PRONTO PARA COMEÇAR?

Deu algum problema no caminho?

Volta para o passo onde travou e releia com calma. Se o problema persistir, o Claude Code mesmo consegue te ajudar — só descreve o erro que apareceu e ele resolve.
